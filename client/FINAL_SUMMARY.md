## 🔄 Authentication Flow

```
User visits application
        ↓
Not authenticated?
        ↓
/auth/role-selection
        ↓
User selects role
        ↓
/auth/login/[role]
        ↓
User enters credentials
        ↓
Form validation
        ↓
API authentication
        ↓
Success? → /dashboard
Error? → Show error message
```
